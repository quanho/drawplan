Hướng dẫn sử dụng DrawPlan

Công cụ vẽ mặt bằng nhà/vườn theo kích thước thực, mô tả bằng JSON.

1. Tổng quan

DrawPlan nhận một tài liệu JSON mô tả mặt bằng, rồi vẽ ra sơ đồ SVG đúng tỉ lệ. Bạn nhập JSON vào ô bên trái, nhấn ▶ Vẽ, sơ đồ hiện ra bên phải.

  • Pan: giữ chuột và kéo trên bản vẽ.
  • Zoom: lăn chuột, hoặc dùng nút + / − / ⊙ góc phải dưới.
  • Xuất HTML: tải về một file HTML độc lập (không cần server) có thể chia sẻ.
  • Copy SVG: chép SVG vào clipboard để dán vào Inkscape, Figma, v.v.

2. Cấu trúc tài liệu JSON

// Ví dụ tối giản
{
  "version": "drawplan/v1",
  "unit": "mm",          // "mm" | "cm" | "m"
  "plot": {
    "width": 60000,     // chiều ngang mảnh đất
    "depth": 22000,     // chiều sâu (dọc)
    "label": "Mảnh đất"
  },
  "dimensionOffset": 0,  // khoảng cách px từ hình đến đường kích thước
  "items": [ /* danh sách các hình */ ]
}
TrườngKiểuBắt buộcMô tả
versionstringkhôngPhiên bản schema, hiện tại "drawplan/v1"
unit"mm" / "cm" / "m"khôngĐơn vị mặc định cho tất cả kích thước. Mặc định: "mm"
plot.widthnumberbắt buộcChiều rộng mảnh đất
plot.depthnumberbắt buộcChiều sâu mảnh đất
plot.labelstringkhôngNhãn hiển thị trên mảnh đất
dimensionOffsetnumberkhôngKhoảng cách (px canvas) từ cạnh hình đến đường kích thước. Mặc định: 0
itemsarraykhôngDanh sách các hình: rect, path, measure

3. Hình chữ nhật — type: "rect"

{
  "id": "house",
  "type": "rect",
  "label": "Nhà",
  "x": 31465,          // tọa độ góc trái trên
  "y": 4500,
  "width": 21000,
  "height": 8730,
  "style": "solid",     // "solid" | "dashed"
  "fill": "#e0f2fe",    // màu nền (CSS color)
  "stroke": "#1a7a3e", // màu viền
  "strokeWidth": 400,  // độ dày nét (cùng đơn vị với "unit")
  "labelRotation": 0,  // xoay nhãn, độ (0 = ngang)
  "showDimensions": true // xem mục 5
}
TrườngKiểuMô tả
x, ynumberTọa độ góc trái trên (cùng đơn vị unit)
width, heightnumberKích thước
style"solid" / "dashed"Kiểu nét viền. Mặc định: "solid"
fillCSS color stringMàu nền. Mặc định: không tô ("none")
strokeCSS color stringMàu viền. Mặc định: "#1a7a3e"
strokeWidthnumberĐộ dày nét tính theo đơn vị unit. Ví dụ: đơn vị mm, 400 ≈ 4px trên canvas
labelRotationnumber (độ)Xoay nhãn. 90 = dọc. Nếu bỏ qua, tự động theo cạnh dài hơn
showDimensionsxem mục 5Hiển thị đường kích thước

4. Đường gấp khúc — type: "path"

{
  "id": "fence",
  "type": "path",
  "label": "Hàng rào",
  "points": [
    [0, 0], [10000, 0], [10000, 5000]
  ],
  "stroke": "#94a3b8",
  "strokeWidth": 200,
  "style": "dashed"
}

points là mảng các điểm [x, y]. Vẽ các đoạn thẳng nối nhau, không tự đóng.

5. Đường đo khoảng cách — type: "measure"

{
  "id": "gap1",
  "type": "measure",
  "from": [0, 1125],
  "to":   [1125, 1125],
  "label": "1125 mm",  // bỏ qua để tự tính
  "offset": 0,           // dịch chuyển vuông góc (px)
  "stroke": "#888",
  "labelPos": "outer"   // "outer" (trên đường) | "inner" (dưới đường)
}

Vẽ đường mũi tên hai đầu giữa hai điểm tùy ý. Nếu bỏ label, khoảng cách được tính tự động theo đơn vị của tài liệu.

TrườngKiểuMô tả
from, to[x, y]Hai điểm đầu và cuối
labelstringNhãn tuỳ chỉnh. Bỏ qua để tự tính khoảng cách
offsetnumber (px)Dịch đường đo ra ngoài vuông góc. Mặc định: 0
strokeCSS colorMàu đường và nhãn. Mặc định: "#888"
labelPos"outer" / "inner"Vị trí nhãn so với đường. Mặc định: "outer" (phía trên)

6. Đường kích thước — showDimensions

Áp dụng cho rect. Điều khiển đường kích thước hiện trên cạnh nào và nhãn đặt ở đâu.

Các dạng giá trị

Giá trịÝ nghĩa
true (mặc định)Hiện cạnh top và right, nhãn phía ngoài
falseẨn tất cả đường kích thước
["bottom", "left"]Hiện đúng các cạnh liệt kê, nhãn outer
{ "bottom": true }Merge vào mặc định: top + right + bottom
{ "top": false }Merge vào mặc định: ẩn top → chỉ còn right
{ "bottom": { "labelPos": "outer" } }Merge: thêm bottom với labelPos chỉ định
Lưu ý về record form { }: Luôn bắt đầu từ mặc định top + right, rồi áp override. Dùng false để ẩn một cạnh, true hoặc { labelPos } để thêm/thay đổi cạnh. Dạng array [ ] thay thế hoàn toàn, không merge.

Vị trí nhãn — labelPos

Cạnh"outer" (xa shape)"inner" (gần shape)
topNhãn phía trên đườngNhãn phía dưới đường
bottomNhãn phía dưới đườngNhãn phía trên đường
leftNhãn bên trái đườngNhãn bên phải đường
rightNhãn bên phải đườngNhãn bên trái đường
// Record form — merge vào top+right mặc định
// Thêm bottom (outer), giữ top+right
"showDimensions": { "bottom": { "labelPos": "outer" } }

// Ẩn top, thêm left → chỉ còn right + left
"showDimensions": { "top": false, "left": true }

// 4 cạnh tuỳ chỉnh hoàn toàn bằng array (thay thế, không merge)
"showDimensions": ["top", "left"]

// Ẩn tất cả
"showDimensions": false

8. Nhãn hình chữ nhật — labelRotation

Nhãn tự động xoay theo cạnh dài hơn của hình:

Trường hợpGóc mặc định
width > height (hình ngang)0° — nằm ngang
height > width (hình đứng)90° — xoay dọc

Để override, thêm "labelRotation" với bất kỳ góc nào (kể cả 0 để ép ngang dù hình đứng):

"labelRotation": 0   // ép ngang
"labelRotation": 90  // ép dọc
"labelRotation": 45  // chéo 45°

9. Đơn vị và quy đổi

Tất cả số đo trong JSON dùng chung đơn vị unit khai báo ở gốc tài liệu. Riêng strokeWidth cũng theo đơn vị đó.

unitVí dụ strokeWidth 4pxGhi chú
"mm"~300–500 (tùy tỉ lệ)Bản vẽ rộng 60 m → 1 mm ≈ 0.02 px
"cm"~30–50
"m"~0.3–0.5
Để có nét đậm hơn bình thường, hãy thử tăng strokeWidth và xem kết quả trực tiếp trên bản vẽ.

10. Ví dụ đầy đủ

{
  "version": "drawplan/v1",
  "unit": "mm",
  "plot": { "width": 59360, "depth": 21940, "label": "Mảnh đất" },
  "dimensionOffset": 0,
  "items": [
    {
      "id": "house", "label": "Nhà", "type": "rect",
      "x": 31465, "y": 4500, "width": 21000, "height": 8730,
      "style": "solid", "fill": "#f0fdf4", "strokeWidth": 400,
      "showDimensions": ["top", "right"]
    },
    {
      "id": "garden", "label": "Sân vườn", "type": "rect",
      "x": 0, "y": 0, "width": 31465, "height": 21940,
      "style": "dashed", "fill": "#f0fdf4",
      "showDimensions": {
        "bottom": { "labelPos": "outer" },
        "left":   { "labelPos": "outer" }
      }
    },
    {
      "id": "gap", "type": "measure",
      "from": [0, 0], "to": [0, 1125]
    }
  ]
}
